# Petal — Event Planning Prototype

A lightweight HTML/CSS prototype for **Petal**, an AI-powered wedding and event planning platform aimed at brides, mothers-of-the-bride, and wedding planners.

## Live preview

👉 **https://arctic-byte-as.github.io/event-planner-ai-prototype/**

## Screens

| Page | Description |
|---|---|
| [index.html](index.html) | Landing page — hero, features, testimonials, pricing |
| [dashboard.html](dashboard.html) | Organiser dashboard — event cards, RSVP stats |
| [create-event.html](create-event.html) | 4-step event creation wizard |
| [event-detail.html](event-detail.html) | Event detail — overview, agenda, guests, settings |
| [rsvp.html](rsvp.html) | Public guest RSVP page (mobile-first) |
| [branding.html](branding.html) | Design system — colours, typography, components |

## Brand

- **Name**: Petal
- **Audience**: Brides, mothers-of-the-bride, wedding planners
- **Palette**: Ivory · Blush · Dusty Rose · Champagne · Charcoal
- **Headings**: Playfair Display (serif)
- **Body**: Lato

## Notes

This is a static HTML prototype only — no build step, no frameworks. Open `index.html` directly in a browser or view via GitHub Pages.

The full product spec and source code lives in the private [event-planner-ai](https://github.com/arctic-byte-as/event-planner-ai) repository.
